
1) Когда ты написал несколько файлов в гит ты должен их добавить:
```bash
git add (название файла)
```

либо если хочешь добавить все файлы:
```bash
git add .
```

2)  Всегда после добавления файла ты должен прописывать **комментарий**:
```bash
	git commit -m "(твой комментарий)"
```

3) После того как ты сделал комментарий, пушишь в репозиторий:
```bash
git push origin main
```
